Structure du dépôt

Le dépôt contient plusieurs travaux pratiques réalisés dans le cadre du module **Qualité Logicielle** :

- **tp1-playwright** → Découverte de **Playwright** :  
  Automatisation des tests end-to-end sur l’application TodoMVC.

- **tp-gherkin** → Introduction au **BDD (Behavior Driven Development)** avec **Cucumber + Playwright** :  
  Écriture de scénarios Gherkin et mapping des steps vers du code TypeScript.

## Technologies utilisées
- **Node.js**
- **TypeScript**
- **Playwright**
- **Cucumber**
- **Gherkin**
